ArogyaMitra is an AI-powered fitness and wellness platform that delivers personalized health guidance through intelligent workout planning, nutrition recommendations, and real-time wellness support.

## ✨ Features

### 🤖 AI-Powered Coaching
- **Personalized Workout Plans**: AI-generated plans based on fitness level, goals, and preferences
- **Smart Nutrition Guidance**: Customized meal plans considering dietary restrictions and allergies
- **AROMI AI Coach**: Real-time conversational support for motivation and plan adjustments

### 📊 Comprehensive Dashboard
- Real-time progress tracking
- Workout streak monitoring
- Calorie burn analytics
- Upcoming workout schedule
- Charity impact visualization

### 🎯 Core Functionality
- User authentication with JWT
- Personalized workout generation
- Custom meal planning
- Progress tracking with achievements
- YouTube integration for exercise videos
- Spoonacular API for recipes
- Google Calendar sync (optional)

### 💪 Gamification
- Workout streaks
- Achievement badges
- Charity contributions based on fitness milestones

## 🛠️ Tech Stack

### Backend
- **Framework**: FastAPI
- **Database**: SQLite (development), PostgreSQL (production)
- **AI Integration**: Groq LLaMA-3.3-70B
- **Authentication**: JWT with OAuth2
- **External APIs**:
  - YouTube Data API v3
  - Spoonacular API
  - Google Calendar API

### Frontend
- **Framework**: React 18
- **State Management**: Zustand
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **HTTP Client**: Axios
- **Routing**: React Router v6

## 📋 Prerequisites

- Python 3.11+
- Node.js 18+
- npm or yarn
- Git
- API Keys for:
  - Groq API
  - YouTube Data API
  - Spoonacular API
  - Google Calendar API (optional)
